Beelink Setup Commands

Click each box to select, then copy. Run in PowerShell as Admin on Beelink.

Step 1: Install SSH Server

Add-WindowsCapability -Online -Name OpenSSH.Server~~~~0.0.1.0

Step 2: Start SSH

Start-Service sshd

Step 3: Auto-start on boot

Set-Service -Name sshd -StartupType Automatic

Step 4: Allow SSH through firewall

New-NetFirewallRule -Name sshd -DisplayName 'OpenSSH Server' -Enabled True -Direction Inbound -Protocol TCP -Action Allow -LocalPort 22

After all 4, tell me the Windows username (type whoami in PowerShell)
