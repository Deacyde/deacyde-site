40-Slot Bag UI Fix

Step 1: Download the UI file

In Konsole, run:

curl -o ~/Games/EverQuest/Full_RoF2/uifiles/default/EQUI_ContainerWindow.xml https://dev.deacyde.com/EQUI_ContainerWindow.xml

Step 2: Restart EQ client

Close EQ completely and relaunch. The new UI will load automatically.

Step 3: Buy your 40-slot bags!

Go to Baggins Bagsworth in The Rad.
