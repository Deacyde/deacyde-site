40-Slot Bag UI Fix

Download Defiance UI (has 40-slot bag support):

Defiance RoF2 UI on EQInterface

Or try Vegarlson Asylum UI:

Vegarlson Asylum Custom UI on EQInterface

Install:

1. Download the zip

2. Extract into ~/Games/EverQuest/Full_RoF2/uifiles/ (creates its own folder)

3. In EQ, type /loadskin [foldername] to load it, or select it from Options > General > Load UI Skin
