🌟 Starlight Fresh Setup Instructions

⚠️ Prerequisites: GUI is enabled. Login to Starlight VM console.

Step 1: Open Terminal

In the Starlight VM GUI, open a terminal window.

Step 2: Run Agent Init

cd ~/openclaw
node dist/index.js agents init

Step 3: Follow the Wizard

  • Agent name: starlight
  • Model: Choose OpenAI gpt-4o-mini
  • API key: Paste from terminal below
  • Enable Discord: Yes
  • Discord token: (paste your bot token)

Step 4: After Setup

Come back to Copilot and type: "done"

Copilot will copy personality files and configure tools access.

Step 5: Disable GUI

After Copilot confirms setup is complete, run:

sudo systemctl set-default multi-user.target
sudo systemctl stop gdm3

🔑 Get API Key from Terminal

The API key has been provided in your terminal session for security.

Copy it from there and paste into the wizard.

📁 Backup Location: Personality files are saved at ~/starlight-personality-backup/
