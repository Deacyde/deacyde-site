🌟 GitHub Copilot Auth - Starlight VM

📍 Location: Run these commands on Starlight VM (Beelink) terminal

📋 Commands to Run

1. Set up Node.js path: 
export PATH=$PATH:~/.nvm/versions/node/v22.22.2/bin
2. Navigate to OpenClaw directory: 
cd ~/openclaw
3. Start GitHub Copilot authentication: 
node dist/index.js models auth login-github-copilot

📝 What Happens Next

✅ You'll see a URL and Code displayed in the terminal
🌐 Open the URL in a browser (can be on your main computer)
🔑 Enter the code when prompted
✓ Click "Authorize" on GitHub
✨ Success: You'll see a confirmation message. The auth token is saved automatically!

🚪 After Auth Complete

Just logout or close the terminal - you're done! Copilot will tell me to disable the GUI.

🔗 One-Liner (All Commands)

export PATH=$PATH:~/.nvm/versions/node/v22.22.2/bin && cd ~/openclaw && node dist/index.js models auth login-github-copilot
💡 Tip: Copy the one-liner above and paste it all at once into the terminal!
