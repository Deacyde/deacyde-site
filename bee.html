Beelink SSH Access

Open this page on the Beelink and paste the PowerShell block below into a PowerShell window running as deacyde-minipc. This adds the Fedora machine public key so Copilot can SSH in from this box.

What this does

  1. Installs or verifies the Windows OpenSSH server
  2. Starts sshd and sets it to auto-start
  3. Opens the firewall rule for SSH
  4. Adds this Fedora machine's public key to C:\Users\deacyde-minipc\.ssh\authorized_keys
  5. Fixes file permissions so OpenSSH accepts the key
After I confirm I can log in, this page can be deleted.

PowerShell to run on the Beelink

$pub = 'ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIDbsaKyg1yPB+f3jGRkLPMTDGXvP8VT19hFoUa7E1Lgr deacyde@fedora'

$cap = Get-WindowsCapability -Online | Where-Object Name -like 'OpenSSH.Server*'
if ($cap.State -ne 'Installed') {
    Add-WindowsCapability -Online -Name $cap.Name
}

Start-Service sshd
Set-Service -Name sshd -StartupType Automatic

if (-not (Get-NetFirewallRule -Name 'OpenSSH-Server-In-TCP' -ErrorAction SilentlyContinue)) {
    New-NetFirewallRule -Name 'OpenSSH-Server-In-TCP' `
        -DisplayName 'OpenSSH Server (sshd)' `
        -Enabled True `
        -Direction Inbound `
        -Protocol TCP `
        -Action Allow `
        -LocalPort 22
}

$sshDir = Join-Path $env:USERPROFILE '.ssh'
$keyFile = Join-Path $sshDir 'authorized_keys'

New-Item -ItemType Directory -Force -Path $sshDir | Out-Null
if (-not (Test-Path $keyFile)) {
    New-Item -ItemType File -Force -Path $keyFile | Out-Null
}

$existing = if (Test-Path $keyFile) { Get-Content $keyFile -Raw } else { '' }
if ($existing -notmatch [regex]::Escape($pub)) {
    Add-Content -Path $keyFile -Value $pub
}

icacls $sshDir /inheritance:r | Out-Null
icacls $sshDir /grant:r "$env:USERNAME:(OI)(CI)F" "SYSTEM:(OI)(CI)F" | Out-Null
icacls $keyFile /inheritance:r | Out-Null
icacls $keyFile /grant:r "$env:USERNAME:F" "SYSTEM:F" | Out-Null

Write-Host ''
Write-Host 'Done. SSH key added for:' $env:USERNAME
Write-Host 'Host:' $env:COMPUTERNAME
Write-Host 'IP test target: 192.168.1.66'
