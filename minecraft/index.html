🪧 Admin Signs v0.1.0
Glowing wall signs that trigger admin actions when right-clicked by any player. Place them in lobbies, spawn areas, or admin rooms.
🎮 Controls
Trigger sign effect
Right-clicksign block
Break / remove sign
Left-clickany tool or fist
🎁 Give Signs
💡 What Each Sign Does
Gamemode Sign — Cycles clicker through Survival → Creative → Spectator → Adventure.
Time Sign — Toggles between day and night.
Weather Sign — Cycles Clear → Rain → Thunder.
Heal Sign — Fully restores health and hunger of the clicker.
Give Sign — Gives the clicker a preset item bundle.
Fly Sign — Toggles flight ability for the clicker.
Nether Kit Sign — Gives fire resistance, full armor set, and weapons for Nether survival.
Signs work for any player who right-clicks — designed for admin/spawn rooms. Break with your hand to remove.

💡 Block Tips
Splitter Belt — Distributes items alternately left and right.
Merger Belt — Combines two belt lines into one output.
Detector Belt — Emits a redstone signal when an item passes over it.
Conveyor Chest — Chest that auto-inserts items onto adjacent belts.

🎁 Give Items
🔄 Placement Modes
Replace Air Only — Only places blocks where there is air. Safe for surface builds — existing structures preserved.
Replace Non-Solid — Replaces air and non-solid blocks (grass, flowers, etc.) but keeps solid blocks.
Replace Everything — Overwrites all blocks including air. Use underground — properly carves out interior space.
Cycle modes using Scroll while holding the Schematic Placer.
⚙️ Server Config
placement.maxBlocks — Max blocks per operation (default: 50,000).
placement.blocksPerTick — Blocks placed per server tick (default: 200).
placement.cooldownTicks — Ticks between placements per player (default: 200).
permissions.requiredOpLevel — Min OP level to use the placer (default: 2).
files.folder — Schematics folder path (default: schembuilder/schematics).

Break / retrieve turret
Left-clickany tool
🎯 Target Modes
Hostile Mobs — Zombies, skeletons, creepers, spiders, etc. (default)
Players — All players within range. Use for PvP arenas.
All Entities — Everything including passive mobs and players.
Each turret has an overheat bar — rapid firing triggers a cooldown period.
