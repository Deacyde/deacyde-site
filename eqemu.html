🛡️ Mercenaries
Mercs are hired from Guardian NPCs in PoK (near each racial gate and along the wall by Castle Drewe). Right-click a Guardian to open the merc hire window. Mercs are free on this server (purchase/upkeep costs disabled).

Merc types: Tank (Warrior) and Healer (Cleric)
Tiers: Apprentice (I-V) and Journeyman (I-V) — Journeyman are significantly better
Proficiencies: Each race offers different merc options

Mercs are controlled through the Mercenary window (default hotkey: Alt+M or through the EQ menu). No chat commands needed — everything is done through the UI window.
Hire a Merc
Right-click any Guardian NPC in PoK → Select merc type/tier → Hire
Suspend Merc
Use Mercenary window → Suspend button. Saves merc for later resummon.
Unsuspend Merc
Use Mercenary window → Unsuspend button. Brings merc back.
Change Stance
Mercenary window → Stance dropdown. Controls combat behavior.
Dismiss Merc
Mercenary window → Dismiss button. Permanently releases the merc.
Note: If Guardian NPCs aren't showing in PoK, run #repop poknowledge in-game after zoning to PoK. Their spawn expansion requirement has been set to always spawn regardless of server expansion.
